## How to Contribute
1. Fork โปรเจกต์นี้ไปยังบัญชีของคุณ
2. สร้างสาขาใหม่ (branch) สำหรับฟีเจอร์หรือการแก้ไขที่คุณต้องการทำ
3. ทำการ commit การเปลี่ยนแปลง
4. ส่ง pull request เพื่อให้ทีมตรวจสอบ
